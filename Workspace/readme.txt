Provocarea de saptamana asta este configurati un workspace pentru pragramare c++
1) Faceti un cont pe https://gitlab.ucv.ro/ cu contul institutional. Creati un proiect nou.
2) instalati git pe masina voastra virtuala git si intr-un folder faceti checkout( sau primul pull) la proiectul de la pasul 1. Va trebui sa va descurcati cu Linux
3) instalati git pe calculatorul vostru si intr-un folder faceti checkout la proiectul de la pasul 1. Va trebui sa va descurcati cu windows.
4) creati un program c++ Hello World pe calculatorul vostru (windows) , comit, push pe serverul git de la pasul 1
5) logati-va pe masina virtuala linux si faceti pull. Rulati programul.
